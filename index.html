PROGRAMMING

I'm self-taught in GML (Game Maker Language) and have been using it to make games for over 10 years. I'm also self-taught in the basics of Python - See 'Stock Trader' for more info! I'm currently taking the Coding Skills Bootcamp course with IO Academy where I'm learning HTML, CSS, JS, and BASH, with REACT, PHP and SQL to come. I designed and programmed this website as part of the course.

GAMES

I've been developing games since 2013. I primarily use Unreal Engine, and Game Maker: Studio. In 2014 I attended Norwich University of the Arts to study Games Art and Design where I achieved a first class honours. Neon Parasite, which I exhibited at festivals around England was released in 2020 and is my biggest project to date.

ART

I create 3D artwork using Blender, Procreate, and Photoshop. I love making creatures and characters and I like to walk the line between creepy and cute. I also use traditional mediums such as collage, and pencil/paper to create 2D artwork. I have a degree in Visual Communication from Chesterfield college where I achieved a Merit.
